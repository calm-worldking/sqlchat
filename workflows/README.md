# n8n Workflows

В этой папке вы можете хранить экспортированные n8n workflows в формате JSON.

## Экспорт workflow из n8n

1. Откройте n8n в браузере (http://localhost:5678)
2. Перейдите к нужному workflow
3. Нажмите на меню (три точки) в правом верхнем углу
4. Выберите "Export" -> "Workflow"
5. Сохраните JSON файл в эту папку

## Импорт workflow в n8n

1. Откройте n8n в браузере (http://localhost:5678)
2. Нажмите на кнопку "+ Create workflow" или на меню (три точки) в верхнем правом углу и выберите "Import from file"
3. Выберите JSON файл из этой папки
4. После импорта не забудьте активировать workflow, если это необходимо

## Настройка webhook URL

Для правильной работы с SQL Chat, убедитесь, что webhook настроен на URL:
```
http://localhost:5678/webhook-test/chat
```

## Рекомендации по именованию

Рекомендуется использовать понятные имена файлов, например:
- `sql-chat-main-workflow.json` - основной рабочий процесс для SQL Chat
- `data-visualization-workflow.json` - рабочий процесс для визуализации данных

## Версионирование

Рекомендуется добавлять версию в имя файла, например:
- `sql-chat-main-workflow-v1.0.json`
- `sql-chat-main-workflow-v1.1.json`

Это поможет отслеживать изменения в рабочих процессах.
